## 🤖 AI Features Integration

- **Trigger**: "Generate Summary with AI" button in project form
- **Input**: Project title, description, and tech stack

- **Actions**: Accept, Generate Another, Cancel

### Bio Generation
- **Trigger**: "Generate Bio with AI" button in profile form
- **Input**: User name, current role, skills, tone preference
- **Output**: Professional bio description
- **Actions**: Accept, Generate Another, Cancel

- `POST /api/ai/generate-project-summary` - Generate project summaries
